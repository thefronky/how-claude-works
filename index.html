WHAT YOU JUST SAW

The instructions file is the personality. It's just a text file — who you are, what you sell, your taste, your hard rules. Claude reads it every chat, so you never repeat yourself. Turn OFF "my brand is minimalist nature" and ask for ideas: you get generic mush. That's the whole trick — most of "wow, my AI gets me" is a good instructions file.

Memory is earned, not written. Claude keeps its own notes: what sold, what flopped, which supplier breaks mugs. New chats start smart because old chats wrote things down. Turn OFF "sunflowers flopped twice" and watch Claude cheerfully suggest sunflowers again — an AI without memory repeats every mistake forever.

Rules are seatbelts. "Nothing goes public without my OK" turns a robot that acts into a robot that prepares — it does 95% of the work and holds the last 5% for a human eye. Run the listing scenario with that rule off to see why the seatbelt exists.

Claude searches memory — it doesn't read the whole diary. With hundreds of notes, reading everything every time would be slow and expensive. So memory is organized like a book: an index up front, chapters behind it. A question about mugs opens the supplier chapter — and never touches the design notes.
