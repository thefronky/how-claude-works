THE WHOLE TRICK

The engine never changed. You swapped roles and the green box at the top stayed exactly the same. Every "specialist" is that one model plus a few text files.

An .md is just a text file it reads first. No magic, no training, no code. House rules, notes about you, background reading, and a job description — read before the work starts, every single time.

Two files ride along on every job. House rules and memory are worn by every outfit. The role brief and the reference doc are what make one worker different from the next.

Notice nobody gets to publish. Every role can draft, read, and calculate. None can send, post, or spend. That's deliberate — the last step is yours, and it's the cheapest safety there is.
